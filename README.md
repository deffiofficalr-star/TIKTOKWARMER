# ETONAPY — TikTok Warmer Pro

**Консольный инструмент для автоматического прогрева TikTok-аккаунтов.**

---

## 🧠 Что это?

Это программа, которая **автоматически** выполняет 7-дневный сценарий прогрева TikTok-аккаунта:
- смотрит ленту
- ставит лайки
- пишет комментарии
- заполняет профиль
- выкладывает видео
- отвечает на комментарии
- добавляет ссылку

---

## 🚀 Быстрый старт

```bash
git clone https://github.com/ваш-ник/etonapy.git
cd etonapy
pip install -r requirements.txt
python main.py
```

---

📦 Установка

```bash
pip install selenium undetected-chromedriver
```

---

🔐 Вход

Поддерживаются:

· Email
· Google
· Facebook

---

💾 Сессии

Программа сохраняет сессию (куки) после каждого дня.
При следующем запуске предлагает продолжить с того же дня.

---

📁 Видео

Папка videos/ создаётся автоматически.
Сюда нужно класть видео для публикации (форматы: .mp4, .mov, .avi).

---

📅 Сценарий прогрева

День Действие
1 Лента 15–20 мин, лайки, комментарии
2–3 Лента 20–25 мин, лайки, комментарии
4–5 Заполнение профиля + 1 видео
6–7 1–2 видео + ответы на комментарии + ссылка
